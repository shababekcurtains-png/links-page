# صفحة الروابط مع Lower Thirds التفاعلية

## 📝 وصف المشروع
صفحة روابط احترافية تجمع بين تصميم Lower Thirds الحديث وعناصر تفاعلية جذابة.

## ✨ المميزات
- تصميم متجاوب مع جميع الأجهزة
- أزرار Lower Thirds تفاعلية مع تأثيرات حركية
- وضع النهاري/الليلي
- إشعارات منبثقة أنيقة
- نافذة عرض خاص
- خلفية متحركة
- معرض أعمال تفاعلي
- شريط أدوات سفلي

## 🚀 كيفية الاستخدام
1. حمل الملفات الثلاثة (index.html, style.css, script.js)
2. ضعها في مجلد واحد
3. افتح ملف index.html في المتصفح
4. عدل الروابط والصور لتناسب احتياجاتك
